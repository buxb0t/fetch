' Create a WScript shell object to show message boxes Set WshShell = CreateObject("WScript.Shell") ' Prompt for username username = InputBox("Enter your username:", "Login Page") ' Prompt for password password = InputBox("Enter your password:", "Login Page") ' Simulate a login check (this example just accepts any input) If username <> "" And password <> "" Then ' Display a success message WshShell.Popup "Login successful! Welcome, " & username & "!", 5, "Login Status", 64 Else ' Display an error message WshShell.Popup "Login failed! Please enter valid credentials.", 5, "Login Status", 48 End If ' Clean up Set WshShell = Nothing
